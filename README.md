# 使用的技术

1、Vue 2.5.10  
2、ES6          
3、Sass       
4、element-ui 2.3.4      
5、Axios       
6、Echart     
7、其他辅助技术包括Webpack、Gulp、Node等

## 浏览器兼容性     

兼容全球使用率大于1%的浏览器     
兼容每个浏览器的最近两个版本     
不兼容ie8及以下,不完全兼容ie9，兼容ie11及以上     
Edge16及以上    
火狐Firefox52及以上    
兼容谷歌Chrome     

## 安装运行打包

1、npm install         
2、npm run dev           
3、npm run build
